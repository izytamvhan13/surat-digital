# Surat Digital

Aplikasi **Sistem Pengelolaan Surat Digital** berbasis web yang dibangun menggunakan framework **Laravel**.  
Aplikasi ini digunakan untuk membantu proses pengelolaan surat masuk dan surat keluar secara terstruktur, terdokumentasi, dan mudah dipantau oleh administrator maupun pimpinan.

## Teknologi yang Digunakan

-   Laravel
-   PHP
-   MySQL
-   HTML, CSS, JavaScript

## Tentang Laravel

Laravel adalah framework aplikasi web dengan sintaks yang ekspresif dan elegan, yang mempermudah pengembangan aplikasi melalui fitur routing, ORM Eloquent, migration database, dan sistem keamanan yang baik.

Framework ini bersifat open-source dan dilisensikan di bawah lisensi MIT.
